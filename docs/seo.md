# Technical Checklist

Every page must

- Generate dynamic metadata
- Include canonical URL
- Generate structured data
- Use semantic HTML
- Optimize images
- Lazy load below-the-fold assets
- Use descriptive URLs
- Pass Core Web Vitals
- Be mobile friendly
- Avoid duplicate content

---

# Content Checklist

Every page should

- Target one primary keyword
- Include related semantic keywords
- Use H1 only once
- Maintain heading hierarchy
- Include internal links
- Include descriptive alt text
- Answer user intent
- End with a clear CTA

---

# SEO Anti-Patterns

Never

- Duplicate titles
- Duplicate meta descriptions
- Keyword stuffing
- Hidden text
- Thin content
- Auto-generated pages without value
- Broken internal links
- Large unoptimized images
- Multiple H1 tags
- Empty alt attributes for product images
- Index cart, checkout, account, or search result pages

---
